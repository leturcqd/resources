 \subsection*{Exercice 1}

Dire si : \begin{enumerate}
\item $15$ est multiple de $3$, 
\item $15$ divise $60$, 
\item $15$ divise $3$, 
\item $15$ est divisible par $3$. 
\end{enumerate}
\subsection*{Exercice 2}
 Effectuer les divisions euclidiennes de 
 \begin{enumerate}
 \item $151$ par $3$
 \item $151$ par $7$
 \item $793$ par $7$
 \item $1262$ par $15$
 \item $408$ par $13$ 
 \item $229$ par $17$
 \item $135$ par $24$
 \item $6416$ par $12$
 \item $2023$ par $31$
 \end{enumerate}
